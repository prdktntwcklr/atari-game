# Atari 2600 Game

A game for the Atari 2600. Based on Pikuma's *Atari 2600 Programming with
6502 Assembly* course. Run ```make``` from the top-level directory to build the
project.

## Toolchain

- DASM 2.20.14.1
- GNU Make 4.2.1
