# AtariGame

A game for the Atari 2600 system. Based on Pikuma's *Atari 2600 Programming with 6502 Assembly* course.

## Toolchain
- DASM 2.20.14.1
- GNU Make 4.2.1

## Getting Started
To compile the project, run the following command from the home directory:
```
$ make
```
